# business-payment-analysis-location-dashboard.

This project is a Power BI dashboard created to analyze student payments, course fees, and revenue trends across different institutes and locations.

---

## 📊 Features

* KPI Cards for Total Course Fees, Total Paid Fees, Student Count, and Average Paid Fees
* Year-wise comparison of Course Fees vs Paid Fees
* Payment Mode analysis (Classroom, Online, Fast Track, Customized)
* Student Stage Analysis (Enquiry → Demo → Confirmation → Payment)
* Trend analysis using line charts
* Summary table with yearly difference calculation
* Interactive filters for Year and Location

---

## 🛠 Tools Used

* Power BI
* Excel
* DAX

---

## 📂 Dataset

* Excel-based business payment dataset
* Fields: Year, Course Fees, Paid Fees, Student ID, Mode, Stage Name, Location
